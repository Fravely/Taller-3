Instrucciones:
- Para el caso de árboles el código se puede correr con el comando "go run main.go" y en el caso del test con "go test"
- Para el caso de colas se necesitará usar el siguiente comando "go run main.go mini.log.txt 5 60" siendo 5 es el máximo de peticiones permitidas por IP y 60 es la ventana de tiempo en segundos, y en el caso del test con "go test"
- Para el caso de listas el código se puede correr con el comando "go run lru.go" y en el caso del test con "go test"
- Para el caso de pilas el código se puede correr con el comando "go run Pilas.go" y en el caso del test con "go test"
